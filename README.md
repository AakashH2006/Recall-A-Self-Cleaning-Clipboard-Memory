# Recall-A-Self-Cleaning-Clipboard-Memory
Recall is a browser extension that acts as a temporary memory for everything you copy

When a user copies text, links, code, or notes, the item is automatically saved in a Recent section with a simple, auto-generated title (e.g., GitHub Repo, Meeting Notes, Python Script). Each item is stored for 24 hours, after which it is automatically deleted to keep the clipboard history clean and clutter-free.

Users can expand any item to view a preview, see its remaining time before expiry, and choose to copy, save, or delete it. If an item is copied again before expiry, the extension detects the duplicate and offers to reset its 24-hour timer instead of creating another entry.

For important items, users can move them to a Vault, a limited-capacity permanent storage area. The Vault displays the number of stored items and encourages users to save only what matters. To prevent the Vault from becoming cluttered, items that remain unused for more than a week trigger a notification asking whether they should be kept or removed (or archived).

Key Features
1. Automatic clipboard history with 24-hour expiry
2. Smart auto-generated titles with manual renaming
3. Expandable previews of copied content
4. Duplicate detection with timer reset
5. Vault for permanently saved items
6. One-time reminder before an item expires
7. Inactivity reminders for old Vault items
8. Privacy-focused, local storage, no AI or external APIs
9. Core Value Proposition

"Your clipboard remembers what's important and forgets what isn't."

Instead of becoming an endless list of copied items, Recall automatically organizes, cleans, and preserves only the content users actually care about.
